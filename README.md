# CandyMark — 移动端浮动书签导航

一个用原生 JavaScript 编写、面向移动端优化的浮动书签导航**用户脚本**（Tampermonkey / Greasemonkey）。在页面上提供可拖拽的悬浮书签，用于快速跳转常用页面；并提供一组基于页面事件的便捷导航封装。

> ⚠️ **仅供学习与技术交流使用**
> 本项目是个人技术练习，开源用于交流前端 / 用户脚本相关技术。请勿用于商业或任何违规用途。
> 其中的「辅助设置」依赖第三方站点的页面行为，使用此类脚本可能与对应服务的使用条款相冲突，**是否使用、以及由此产生的一切后果均由使用者自行承担**，作者不作任何担保、不承担任何责任。

## 功能概览

### 📌 浮动书签导航（核心）
书签本身与具体站点无关，是一套通用的悬浮导航控件：

- **悬浮书签**：页面上的小方块书签，点击即跳转到对应链接
- **添加**：点击左上角触发区域，或快捷键 `Ctrl + B`
- **菜单**：移动端长按（600ms）、PC 端右键打开
- **拖拽定位**：书签可拖到页面任意位置，位置本地保存
- **视觉定制**：10 级大小、10 级透明度，实时预览；内置多组渐变配色循环
- **本地存储**：所有数据存于浏览器 `localStorage`，不上传、不收集

### 🔗 特殊链接
书签的「链接」除普通 URL 外，还支持几种动作：

- **后退**：触发浏览器 `history.back()`
- **刷新**：重新加载当前页
- **穿透点击后退**：点击事件穿透到下层元素，再延迟触发后退（延迟可调）

### 🧭 辅助设置（进阶）
脚本可监听页面的网络响应，在特定事件发生后执行**一个**导航动作（后退 / 刷新 / 跳转到指定书签）：

- **常态**：战斗结束后、结算后
- **战斗内（按副本分别配置）**：前 N 回合攻击后、第 N 回合攻击后、召唤后、技能后
  - 召唤 / 技能可按具体对象过滤（只在选定的召唤石 / 技能上触发）
  - 不同副本各自独立设置，副本以大厅图选择
- **跳转目标**：从已有书签中指定一个作为跳转落点（全局共用）

> 该功能针对碧蓝幻想（Granblue Fantasy）单页应用的页面事件设计，是对页面导航的便捷封装，**仅作技术演示与学习交流**。

### 🔔 掉落提醒
从副本掉落预览页或战斗结算页勾选要关注的物品；当对应物品出现时弹窗提示。

### ⚙️ 配置与个性化
- **导入 / 导出**：完整配置可导出到文件或剪贴板，并从文件 / 剪贴板导入，便于备份与跨设备迁移
- **运行范围**：支持站点黑名单、触发器显隐、快捷键等设置

## 适用范围

脚本的 `@match` 当前配置为碧蓝幻想相关域名：

```
*://game.granbluefantasy.jp/*
*://gbf.game.mbga.jp/*
```

「浮动书签导航」部分本身与站点无关，如需在其它站点使用，可自行调整 `@match`；而「辅助设置 / 掉落提醒」依赖上述站点特定的页面结构，仅在其页面内有意义。

## 安装

1. 安装用户脚本管理器
   - Chrome / Edge：[Tampermonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
   - Firefox：[Tampermonkey](https://addons.mozilla.org/firefox/addon/tampermonkey/)
   - iOS Safari：[Tampermonkey（App Store）](https://apps.apple.com/app/tampermonkey/id1482490089)
2. 安装脚本：[candymark.user.js](https://unixliang.github.io/candymark/candymark.user.js)
3. 打开目标页面，脚本即生效

## 使用速览

| 操作 | 方式 |
|------|------|
| 添加书签 | 点击左上角触发区域 / `Ctrl + B` |
| 打开书签菜单 | 长按书签（移动端）/ 右键（PC） |
| 拖拽移动 | 书签菜单 → 拖拽移动 |
| 设为当前页 / 后退 / 刷新 / 穿透后退 | 书签菜单内对应项 |
| 全局配置 | 书签菜单 → ⚙️ 全局配置（打开左上角主菜单） |
| 辅助设置 | 主菜单 → 🌐 辅助设置（常态） / ⚔️ 辅助设置（战斗） |
| 掉落提醒 | 主菜单 → 🔔 掉落通知 |
| 导入导出 | 主菜单 → ⚙️ 配置管理 |

## 数据与隐私

- 所有数据仅保存在浏览器本地 `localStorage`，**不联网上传、不收集任何个人信息**
- 书签数据键名：`candymark-bookmarks-javascript`；其余设置以 `sb_` 前缀存储
- 卸载脚本或清除站点数据即可彻底删除

## 技术

- 原生 JavaScript（ES6+），无第三方依赖
- 移动端触摸优化：长按检测、手势防护、硬件加速
- 性能：事件委托、增量渲染、防抖存储、配置缓存
- 单文件实现，核心为 `CandyMarkManager`（书签 / UI / 菜单）与 `GameDetector`（页面事件监听）
